# 🔊 Text-to-Speech Generator

A simple **Text-to-Speech (TTS)** application built with **Python**, **gTTS (Google Text-to-Speech)**, and **Gradio**.

The application converts user-entered text into natural-sounding speech and returns an MP3 audio file that can be played directly in the browser.


## ✨ Features

- 📝 Convert text into speech
- 🔊 Generate high-quality MP3 audio
- 🌐 Browser-based interface using Gradio
- ⚡ Fast and lightweight
- 📥 Download generated audio
- 🎯 Beginner-friendly project

# 🎯 How to Use

1. Launch the application.
2. Enter any text into the textbox.
3. Click **Submit**.
4. Wait a few seconds.
5. Listen to the generated speech or download the MP3 file.
